# Wittgenstein / Modality Harness — Delivery index

This folder is **`@wittgenstein/agent-contact-text`**: extended narrative primers for coding agents. Normative implementation rules remain in the monorepo root [`AGENTS.md`](../../AGENTS.md) and [`docs/`](../../docs/).

## Files in this package

1. **`01_Build_Book.md`**
   - Main human/developer document: product positioning, architecture, technical paths, work plan, CLI/skill design, evaluation, and execution guidance.
   - **`01_Build_Book.docx`** — Word source of the same material (optional).

2. **`02_AI_Execution_Context.md`**
   - Compact context for Claude Code / Codex / Cursor / OpenCode; pair with root `AGENTS.md`.

3. **`03_Research_Reference_Dossier.md`**
   - Annotated reference pack: articles, papers, repos, and why each matters.

## Recommended usage order

1. Read root [`AGENTS.md`](../../AGENTS.md) and [`docs/architecture.md`](../../docs/architecture.md) first (short, binding).
2. Read `01_Build_Book.md` for full narrative depth.
3. Hand `02_AI_Execution_Context.md` to coding agents as session seed.
4. Use `03_Research_Reference_Dossier.md` when you need citations and rationale.
